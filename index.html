🔒 Gratis con tu cuenta de Google
Conectar con Google Firebase
Firebase es el servicio gratuito de Google para guardar datos en la nube.
Tus tareas estarán disponibles desde cualquier computadora o celular con solo abrir el link.
Paso 1: Entra a console.firebase.google.com
Paso 2: Inicia sesión con tu cuenta de Gmail
¿Ya configuraste Firebase antes? Ir directo a pegar configuración
Crear tu proyecto Firebase
Sigue estos pasos en la página de Firebase. Solo se hace una vez.
1. Haz clic en Crear un proyecto
2. Escribe cualquier nombre (ej: mi-taskboard) → Continuar
3. Deshabilita Google Analytics → Crear proyecto
4. Espera a que termine y haz clic en Continuar
Activar la base de datos:
5. En el menú izquierdo busca Firestore Database
6. Haz clic en Crear base de datos
7. Elige Comenzar en modo de prueba → Siguiente → Habilitar
Obtener la configuración:
8. Haz clic en el ícono ⚙️ (Configuración del proyecto)
9. Baja hasta "Tus apps" → haz clic en </> (Web)
10. Pon cualquier nombre → Registrar app
11. Copia todo el bloque que dice firebaseConfig = { ... }
Pegar tu configuración de Firebase
Pega aquí el bloque firebaseConfig que copiaste de Firebase.
Solo el contenido entre llaves { ... } o todo el bloque, como prefieras.
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "mi-app.firebaseapp.com",
  projectId: "mi-app",
  ...
};
⚠ No se pudo conectar. Revisa que pegaste la configuración correcta.
